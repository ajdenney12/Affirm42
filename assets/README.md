# Assets Directory

This directory should contain your app icons and splash screens.

## Required Assets

### For iOS and Android:
- `icon.png` - App icon (1024x1024px, PNG)
- `adaptive-icon.png` - Android adaptive icon (1024x1024px, PNG)
- `splash.png` - Splash screen image (1284x2778px recommended)
- `favicon.png` - Web favicon (48x48px)

## Icon Guidelines

### icon.png
- Size: 1024x1024 pixels
- Format: PNG with transparency
- Design: Should include padding, will be automatically masked by iOS

### adaptive-icon.png
- Size: 1024x1024 pixels
- Format: PNG with transparency
- Design: Keep important content in the center 660x660px safe zone

### splash.png
- Size: 1284x2778 pixels (or similar aspect ratio)
- Format: PNG
- Design: Center your logo/brand on a solid background color

## Creating Icons

You can use tools like:
- Figma or Adobe Illustrator for design
- [appicon.co](https://appicon.co) for generating all sizes
- [makeappicon.com](https://makeappicon.com) for comprehensive icon sets

## Quick Setup

For now, you can use a simple colored square as a placeholder:
1. Create a 1024x1024 PNG with your brand colors
2. Add the NextSelf logo or app name
3. Save as `icon.png`, `adaptive-icon.png`, and `splash.png`

The app uses a blue-to-pink gradient theme, so consider using those colors.
